# Hi, I'm Lemon

这里是我的 GitHub 角落，也是我在互联网上持续生长的一小块地方。

我会在这里记录项目、实验、学习过程，以及那些值得认真做一遍的想法。
有些仓库是认真完成的作品，有些是中途发芽的实验，也有一些只是为了验证一个念头到底靠不靠谱。
对我来说，写代码不只是完成任务，更像是一种把模糊想法慢慢雕成真实东西的过程。

## About Me

- 喜欢把想法变成真正能跑起来的东西
- 关注软件工程、效率工具、自动化和有趣的产品体验
- 持续学习，也持续动手

我不太把这里当作一份“求职资料”，而更愿意把它当作一份长期更新的个人记录。
它不一定总是完美，也不一定每个项目都要足够成熟才值得放出来。
比起只展示结果，我也想保留过程本身，包括试错、重构、推翻重来，以及某个时刻非常笃定地想把一件事做出来的那股劲。

## What You'll Find Here

- 一些正在做、刚做完，或者以后可能还会继续打磨的项目
- 一些小工具、小实验，以及为了提升效率写下来的脚本
- 一些学习新技术时留下的练习和阶段性成果
- 一些暂时还不成熟，但我不想让它消失的想法

## Things I Enjoy

- 折腾开发环境，让工具真的为人服务
- 做自动化，把重复的事情交给程序处理
- 打磨细节，尤其是那些会影响体验但不容易被一眼看到的部分
- 关注一个产品从“能用”到“好用”之间的距离

## My Approach

我喜欢边学边做，也相信很多理解只能在亲手实现之后才会真正变得具体。
相比“看起来会了”，我更在意“是否真的做出来过”。

很多时候，我会先从一个很小的原型开始，先验证方向，再一点点补上结构、细节和稳定性。
如果一个项目值得继续，我会愿意花时间把它整理得更清楚；如果它不值得，也没关系，至少它帮我确认了一件事。
